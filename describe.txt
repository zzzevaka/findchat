Сайт знакомств с простым функционалом - пользователи предлагают темы для общения. Другие пользотели отвечают на эти темы. Так завязывается диалог.


<b>Мотивация:</b>

Самым популярным вариантом первого сообщения на сайтах знакомств является простое "привет". Это понятно. Что еще можно написать человеку, о котором ничего не знаешь?
Многие такие сообщения просто игнорируют. Многие даже пишут что-то вроде: "Не начинайте с "привет". Придумайте что-то интересное...".

Некоторым людям просто сложно побороть страх быть отвергнутым, поэтому они просто боятся написать интересному человеку.

Когда тема для разговора уже предложена, пользователю будет проще начать диалог. Частично решается проблема завязывания диалога.

Помимо собственно знакомств, это может быть удобно для получения практики переписки на иностранных языках.


<b>Реализовано:</b>

На данный есть полнофункциональный прототип. Можно зайти потрогать - findchat.io.

Техническая реализация:

Frontend: SPA. Основой является React, Redux, React Router.

Backend: основное API написано на python3 + tornado. Ипользуются базы данных PostgreSQL (+ ORM SQLAlchemy), Redis.
Для балансировки и раздачи статики используется NGINX.
Так же есть Node.js Express сервер для серверного рендеринга.
Все это запущено в Docker-контейнерах. Используется git.

Все это написано мной в свобоное время. Привлекал на фрилансе только дизайнера.
